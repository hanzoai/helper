# @zooai/helper

> Sign in to Zoo and use its AI models in Claude Code, Codex, and more.

```bash
npx @zooai/helper
```

The Zoo-branded build of [`@hanzo/helper`](https://github.com/hanzoai/helper) —
identical engine, Zoo defaults (`zoolabs.id`, `api.zoo.ngo`, `zoo-app`). One
command does browser device login, mints your Zoo API key, and points your
coding tools at the Zoo AI cloud.

See the [upstream README](https://github.com/hanzoai/helper) for the full
command reference. License: Apache-2.0.
