# @luxfi/helper

> Sign in to Lux and use its AI models in Claude Code, Codex, and more.

```bash
npx @luxfi/helper
```

The Lux-branded build of [`@hanzo/helper`](https://github.com/hanzoai/helper) —
identical engine, Lux defaults (`lux.id`, `api.lux.network`, `lux-app`). One
command does browser device login, mints your Lux API key, and points your
coding tools at the Lux AI cloud.

See the [upstream README](https://github.com/hanzoai/helper) for the full
command reference. License: Apache-2.0.
